## What Cisco Portfolio Navigator does

Cisco Portfolio Navigator helps teams shape portfolio conversations, connect account opportunities, and transform product discovery into a more organized planning process. It combines graph-style exploration, step-by-step workflows, and export-oriented features so users can move from early analysis to a polished account plan without leaving the browser.

Because it ships as a single-file HTML application, it is straightforward to open, distribute, and tailor for browser-based use. It fits especially well in sales, solution design, and planning scenarios where fast iteration, clear structure, and presentation-ready output are important.

## Key capabilities

- Structured planning flow for portfolio and account work
- Interactive graph views for navigating relationships across the portfolio
- Account planner tools for arranging customer-facing strategy
- BYOK AI assistant support for assisted planning and content generation
- SVG stencil support for flexible visual building blocks
- Port-based linking to connect elements in a clear, diagram-like way
- Template gallery for quickly starting from reusable layouts
- Design score and suggestion engine for iterative refinement
- PoE tracking to help monitor progress across planning items
- Presentation mode for reviewing and sharing work cleanly
- Customer narrative export for moving from plan to client-ready output

## Usage

1. Open Cisco Portfolio Navigator in your browser.
2. Start a new plan or load an existing template from the gallery.
3. Use the graph view and account planner to organize the portfolio story.
4. Add links, SVG stencils, and plan elements as needed.
5. Run the suggestion engine or AI assistant if you want extra support.
6. Switch to presentation mode when you are ready to review or share.
7. Export the customer narrative for a cleaner handoff or follow-up.
